Preserve — PII detection demo

Detects and scrubs personally identifiable information, then maps it to reversible placeholders.

GitHub repo Benchmarks

🔒
100% private — your text never leaves this browser. No network requests, no server, no logging. Detection runs entirely in client-side JavaScript on your device. (Open your browser's network tab and watch — nothing is sent.)
This demo implements the deterministic detection layers: 53 regex patterns (Layer 2a) across 15+ countries, 9 checksum validators (Layer 2c: Luhn, IBAN, CPF, DNI/NIE, BSN, NHS, HETU, RRN), and context scoring (Layer 2d). Names are caught when they carry a title or label (e.g. “Dr. Lee”, “Patient: Aurora Rossi”); bare names in free text need the gazetteer name scorer or local LLM review (Layers 2h/3), which require the Python package.
