What this does

Detects and scrubs personally identifiable information from text, then maps every redaction to a reversible placeholder like [EMAIL_1]. Type or paste on the right and watch detection happen live.

🔒 Fully private

Your text never leaves this browser. No network requests, no server, no logging. Everything runs in client-side JavaScript on your device. (Open the network tab and watch: nothing is sent.)

How detection works

  • 2a 53 regex patterns across 15+ countries
  • 2c 9 checksum validators (Luhn, IBAN, CPF, DNI/NIE, BSN, NHS, HETU, RRN)
  • 2d context scoring to weight confidence
  • 2h name gazetteer catching bare names across 55 countries

The full Python package adds an optional local LLM (Layer 3) for the very hardest cases. Everything else above runs right here.
