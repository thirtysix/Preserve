Preserve — PII detection demo

Detects and scrubs personally identifiable information, then maps it to reversible placeholders.

GitHub repo Benchmarks

This demo runs entirely in your browser — nothing is sent anywhere. It implements the deterministic detection layers only: 52 regex patterns (Layer 2a) across 15+ countries and 9 checksum validators (Layer 2c: Luhn, IBAN, CPF, DNI/NIE, BSN, NHS, HETU, RRN). The full tool adds a gazetteer-based name scorer and an optional local LLM (Layer 3) that catch contextual PII like bare lowercase names — those require the Python package.
